# Setup

## Requirements

- macOS
- Xcode
- Unity with iOS Build Support
- Apple Developer account or local signing setup
- Physical iOS device recommended

## Setup Steps

1. Create the native iOS app inside `ios/`.
2. Create the Unity project inside `unity/`.
3. Build the Unity project for iOS.
4. Copy UnityFramework artifacts into the native iOS app.
5. Configure Xcode build settings.
6. Run the app on a physical iOS device.
